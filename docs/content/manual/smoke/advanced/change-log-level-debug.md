title: Change log level debug	
---
1. Log in as admin
1. Navigate to advanced settings
1. Edit config on log-level
1. Choose Debug
1. Save
1. Create two VMs
1. Reboot both VMs
1. Download Logs

## Expected Results
1. Login should complete
1. Settings should save
1. VMs should create
1. VMs should reboot sucessfully
1. Logs should show Debug level output
